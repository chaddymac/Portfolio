Below you will find a few Python projects I've created to demonstrate my knowledge and skills.

1. Translator: The translator project uses the Python Translate library to convert my english input into espanol. When using the Translate library in python you are able to translate any language as long as you adjust the code considerably. The translator also has the ability to save the translation to a txt document which I created. Translator

2. Was your password hacked?: The password hacking project uses Python to hash your password so it is not traced. Once the password is hashed it runs it through a database to check if your password has ever been hacked. Hacked Password?

3. Hangman: My personal favorite project! My hangman project I used python to create the famous game of hangman. The program generates a random word then let the fun begin! The game reveals the length of the word without giving any hits. You continuously guess letters and each time you get it correct it will reveal the letter. Translator

4. Password Checker: The password checker uses the regular expressions library to check if your password you give the terminal is a strong. This project was pretty challenging in the beginning because regular expressions can be difficult. With a little research I found a wonderful site which generates your regular expressions for you! Thank you regex101.com you make my life easier. Password Checker

5. Python SMS: The Python SMS project uses the Twilio communication API to generate a message to myself. This project is neat because it can allow a person to automatically send messages to multiple people, send messages at specific time and You can send important messages to others if you are ever simply glued to your computer and too lazy to walk over and grab your phone.
